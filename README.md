A simple desktop-based Contact Management System built using **Java Swing (GUI)**.
This application allows users to add and delete contacts with a name and phone number.

 

###  Features

* Add contacts (Name + Phone number)
* Display contacts in a list
* Delete selected contact from the list
* User-friendly graphical interface using Swing components
* Message dialogs for success and delete confirmation



### Technologies / Tools Used

| Technology                         | Purpose              |
| ---------------------------------- | -------------------- |
| Java (JDK)                         | Programming Language |
| Swing / AWT                        | Building GUI         |
| JFrame, JPanel, JList, JOptionPane | UI Components        |




###  How to Run the Program

####  Run from Command Prompt / Terminal**

1. Open terminal/command prompt in project folder.
2. Compile the program:

   ```sh
   javac Main.java
   ```
3. Run the compiled program:

   ```sh
   java Main
   ```


### How It Works

* When the user enters a contact name and number, and clicks **Add Contact**,
  it will be displayed in the list.
* Selecting a contact and clicking **Delete Contact** removes it from the list.
